# DWM-Project
corporate sustainability analysis using K-Means, Apriori, and Random Forest with data preprocessing and visualization
The dataset consists of survey responses related to:

Sustainability purchasing behavior
Productivity impact of sustainability initiatives
AI adoption challenges
SME sustainability barriers
Government and corporate influence
Textual responses on sustainability challenges

Technologies & Tools
Python
Pandas, NumPy – Data processing
Matplotlib, Seaborn – Visualization
Scikit-learn – Machine Learning
mlxtend – Apriori algorithm
NLTK / Gensim – NLP (LDA Topic Modeling)

Project Workflow
1. Data Preprocessing
Handled missing values
Encoded categorical variables
Prepared dataset for analysis
2. Exploratory Data Analysis (EDA)
Correlation heatmaps
Distribution plots
Identified relationships between sustainability factors
3. Clustering (K-Means)
Grouped respondents into clusters based on sustainability behavior
Used Elbow Method to determine optimal clusters
Key Insight:
Identified three groups:
High sustainability adoption
Moderate adoption (needs support)
Low adoption
4. Association Rule Mining (Apriori)
Discovered relationships between sustainability factors
Key Insight:
Sustainability-related opinions are interconnected
Strong associations between SME support and industry sustainability focus
5. Classification (Random Forest)
Predicted productivity impact based on sustainability-related features
Key Insight:
Sustainable purchasing behavior is the most important factor affecting productivity
6. NLP (Topic Modeling - LDA)
Extracted themes from textual responses
Key Insight:
Identified key challenges and themes in sustainability adoption
Results & Insights
Sustainability adoption varies across different groups
Behavioral patterns influence sustainability engagement
Strong relationships exist between different sustainability factors
Sustainability practices have a measurable impact on productivity
🚀 Key Takeaways
Sustainability is data-driven and measurable
Different groups require different strategies
AI and analytics can help improve sustainability adoption
